## 部署建议

### 生产环境部署前必须完成

1. **安全配置**
   ```bash
   # 生成强随机密钥
   export JWT_SECRET=$(openssl rand -hex 32)
   export SERVICE_SECRET=$(openssl rand -hex 32)
   export CONFIG_ENCRYPTION_KEY=$(openssl rand -hex 32)
   ```

2. **修改默认密码**
   - MySQL root 密码
   - Grafana admin 密码
   - Superset admin 密码

3. **配置 CORS**
   ```bash
   export ALLOWED_ORIGINS=https://your-domain.com
   ```

4. **启用 HTTPS**
   - 配置 SSL 证书
   - 设置 `COOKIE_SECURE=true`

### 部署步骤

```bash
# 1. 拉取代码
git clone <repository>
cd one-data-studio-lite

# 2. 配置环境变量
cp .env.example .env.production
# 编辑 .env.production

# 3. 一键部署
make deploy

# 4. 验证部署
make status
curl http://localhost:8010/health/all
```
