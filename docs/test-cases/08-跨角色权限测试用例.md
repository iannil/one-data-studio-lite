> **文档版本**: v2.0
> **更新日期**: 2026-01-31
> **适用角色**: 所有角色

## 1. 数据访问权限隔离测试

### TC-XROLE-04-06-01: VW 无法写入数据 🔴

**目的**: 验证查看者无法写入数据

**前置条件**:
- 已登录查看者账号

**测试步骤**:
```bash
TOKEN=$(curl -s -X POST http://localhost:8010/auth/login \
  -H "Content-Type: application/json" \
  -d '{"username": "viewer01", "password": "view123"}' | jq -r '.token')

curl -X POST http://localhost:8014/api/data/test_data/write \
  -H "Authorization: Bearer $TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"records": [{"id": 1, "value": "test"}]}'
```

**预期结果**:
- 状态码: 403
- 响应包含: 权限不足
