# 系统管理员测试用例

**角色代码**: ADM
**职责**: 平台运维、用户管理、权限配置、系统监控
**使用服务**: Portal, Cube-Studio, Audit Log

---

## 生命周期阶段

```
入职 → 账号创建 → 权限配置 → 平台部署 → 用户管理 → 系统监控 → 审计管理 → 离职交接
```

---

## 1. 账号初始化阶段

### TC-ADM-01-01: 管理员首次登录 🔴

**目的**: 验证管理员可以使用初始账号登录系统

**前置条件**: Portal 服务已启动

**测试步骤**:
```bash
curl -X POST http://localhost:8010/auth/login \
  -H "Content-Type: application/json" \
  -d '{"username": "admin", "password": "admin123"}'
```

**预期结果**:
- 状态码: 200
- 响应包含:
  - `success`: true
  - `token`: 有效的 JWT
  - `user.role`: "admin"
  - `user.display_name`: "管理员"

---

### TC-ADM-01-02: 获取当前用户信息 🟡

**目的**: 验证可以通过 Token 获取当前用户信息

**前置条件**: 已登录获取 Token

**测试步骤**:
```bash
TOKEN=$(curl -s -X POST http://localhost:8010/auth/login \
  -H "Content-Type: application/json" \
  -d '{"username": "admin", "password": "admin123"}' | jq -r '.token')

# 解码 Token 查看用户信息
echo $TOKEN | cut -d'.' -f2 | base64 -d 2>/dev/null | jq
```

**预期结果**:
- Token payload 包含:
  - `sub`: "admin"
  - `username`: "admin"
  - `role`: "admin"
  - `exp`: 未过期的时间戳
