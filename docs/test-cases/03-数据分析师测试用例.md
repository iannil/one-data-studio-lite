### TC-ANA-01-01: 查询数据集数据 🔴

**目的**: 验证数据分析师可以查询数据集的内容

**前置条件**: Data API 服务已启动，数据库中存在测试数据

**测试步骤**:
```bash
TOKEN=$(curl -s -X POST http://localhost:8010/auth/login \
  -H "Content-Type: application/json" \
  -d '{"username": "admin", "password": "admin123"}' | jq -r '.token')

curl -X GET "http://localhost:8014/api/data/test_users?page=1&page_size=10" \
  -H "Authorization: Bearer $TOKEN"
```

**预期结果**:
- 状态码: 200
- 响应包含:
  - `dataset_id`: "test_users"
  - `total`: 数据总行数
  - `page`: 当前页码
  - `data`: 数据记录数组

**响应示例**:
```json
{
  "dataset_id": "test_users",
  "total": 1000,
  "page": 1,
  "data": [
    {"id": 1, "name": "张三", "email": "zhangsan@example.com"},
    {"id": 2, "name": "李四", "email": "lisi@example.com"}
  ]
}
```

**响应示例**:
```json
{
  "success": true,
  "question": "查询所有用户的姓名和邮箱",
  "generated_sql": "SELECT name, email FROM users",
  "explanation": "此查询从 users 表中选取所有用户的姓名和邮箱字段",
  "columns": ["name", "email"],
  "rows": [["张三", "zhangsan@example.com"], ["李四", "lisi@example.com"]],
```
