# 数据分析师测试用例

> **文档版本**: v2.0
> **更新日期**: 2026-01-31

> **用例数量**: 30

本文档包含数据分析师 (Data Analyst) 的全生命周期测试用例。

## 用户生命周期流程

```
数据探索 → 数据查询 → 报表制作 → 可视化分析 → 洞察分享 → 预警设置
```

## 1. 账号创建阶段

### TC-ANA-01-01-01: 数据分析师账号创建 🔴

**目的**: 验证数据分析师账号创建流程正常

**前置条件**:
- Portal 服务已启动
- 已存在 analyst 角色

**测试步骤**:
```bash
# 由管理员创建
curl -X POST http://localhost:8010/api/users \
  -H "Authorization: Bearer $ADMIN_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "username": "analyst01",
    "password": "ana123",
    "role": "analyst",
    "display_name": "数据分析师",
    "email": "analyst01@example.com"
  }'
```

**预期结果**:
- 状态码: 201
- 用户创建成功

### TC-ANA-01-01-03: 数据分析师修改密码 🟡

**目的**: 验证数据分析师可以修改自己的密码

**前置条件**:
- 已登录数据分析师账号
