# 数据科学家测试用例

> **文档版本**: v2.0
> **更新日期**: 2026-01-31
> **适用角色**: SCI (数据科学家)
> **用例数量**: 35

本文档包含数据科学家 (Data Scientist) 的全生命周期测试用例。

---

## 用户生命周期流程

```
入职 → 账号创建 → 环境配置 → 数据访问 → 模型开发 → 实验管理
    ↓
模型部署 → 结果验证 → 离职
```

---

## 1. 账号创建阶段

### TC-SCI-01-01-01: 数据科学家账号创建 🔴

**目的**: 验证数据科学家账号创建流程正常

**前置条件**:
- Portal 服务已启动
- 已存在 data_scientist 角色

**测试步骤**:
```bash
# 由管理员创建
curl -X POST http://localhost:8010/api/users \
  -H "Authorization: Bearer $ADMIN_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "username": "scientist01",
    "password": "sci123",
    "role": "data_scientist",
    "display_name": "数据科学家",
    "email": "scientist01@example.com"
  }'
```

**预期结果**:
- 状态码: 201
- 用户创建成功
