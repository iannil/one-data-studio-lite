# 管理员测试用例

> **文档版本**: v2.0
> **更新日期**: 2026-01-31
> **适用角色**: ADM (管理员)
> **用例数量**: 38

本文档包含管理员 (Admin) 的全生命周期测试用例。

---

## 用户生命周期流程

```
入职 → 账号创建 → 权限配置 → 平台部署 → 用户管理 → 系统监控 → 审计管理 → 离职交接
```

---

## 1. 账号创建阶段

### TC-ADM-01-01-01: 管理员账号创建 🔴

**目的**: 验证管理员账号创建流程正常

**前置条件**:
- Portal 服务已启动
- 超级管理员已创建管理员角色

**测试步骤**:
```bash
# 由超级管理员创建管理员账号
curl -X POST http://localhost:8010/api/users \
  -H "Authorization: Bearer $SUPER_ADMIN_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "username": "admin01",
    "password": "admin123",
    "role": "admin",
    "display_name": "系统管理员",
    "email": "admin01@example.com"
  }'
```

**预期结果**:
- 状态码: 201
- 用户创建成功

---

### TC-ADM-01-01-02: 管理员首次登录 🔴

**目的**: 验证管理员可以正常登录

**前置条件**:
- 管理员账号已创建

**测试步骤**:
```bash
curl -X POST http://localhost:8010/auth/login \
  -H "Content-Type: application/json" \
  -d '{"username": "admin01", "password": "admin123"}'
```

**预期结果**:
- 状态码: 200
- 响应包含:
  - `user.role`: "admin"
  - `token`: 有效的 JWT token

---

### TC-ADM-01-01-03: 管理员修改密码 🟡

**目的**: 验证管理员可以修改自己的密码

**前置条件**:
- 已登录管理员账号

**测试步骤**:
```bash
TOKEN=$(curl -s -X POST http://localhost:8010/auth/login \
  -H "Content-Type: application/json" \
  -d '{"username": "admin01", "password": "admin123"}' | jq -r '.token')

curl -X POST http://localhost:8010/auth/change-password \
  -H "Authorization: Bearer $TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "old_password": "admin123",
    "new_password": "NewSecurePass123!"
  }'
```

**预期结果**:
- 状态码: 200
- 密码修改成功

---

### TC-ADM-01-01-04: 验证管理员权限范围 🔴

**目的**: 验证管理员拥有预期的权限

**前置条件**:
- 已登录管理员账号

**测试步骤**:
```bash
TOKEN=$(curl -s -X POST http://localhost:8010/auth/login \
  -H "Content-Type: application/json" \
  -d '{"username": "admin01", "password": "admin123"}' | jq -r '.token')

  -H "Authorization: Bearer $TOKEN"
```

**预期结果**:
- 状态码: 200

  - `data:read`, `data:write`

  - 不包含 `system:admin`、`sensitive:manage`

### TC-ADM-07-01-01: 管理员账号禁用流程 🟡

**目的**: 验证管理员账号被禁用后无法操作

**前置条件**:
- 已登录管理员账号
- 由超级管理员禁用该账号

**测试步骤**:
```bash
# 1. 获取管理员 Token
TOKEN=$(curl -s -X POST http://localhost:8010/auth/login \
  -H "Content-Type: application/json" \
  -d '{"username": "admin01", "password": "admin123"}' | jq -r '.token')

# 2. 超级管理员禁用该管理员
SUPER_TOKEN=$(curl -s -X POST http://localhost:8010/auth/login \
  -H "Content-Type: application/json" \
  -d '{"username": "super_admin", "password": "admin123"}' | jq -r '.token')

curl -X POST http://localhost:8010/api/users/admin01/disable \
  -H "Authorization: Bearer $SUPER_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"reason": "离职", "disabled_by": "super_admin"}'

# 3. 尝试使用被禁用的管理员 Token
curl -X GET http://localhost:8010/api/users \
  -H "Authorization: Bearer $TOKEN"
```

**预期结果**:
- 状态码: 401 或 403
- 响应包含: 账号已被禁用或 Token 无效
