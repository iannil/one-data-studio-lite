# 查看者测试用例

> **文档版本**: v2.0
> **更新日期**: 2026-01-31
> **适用角色**: VW (查看者)
> **用例数量**: 20

本文档包含查看者 (Viewer) 的全生命周期测试用例。

---

## 用户生命周期流程

```
账号创建 → 权限分配 → 数据浏览 → 报表查看 → 导出申请 → 权限回收
```

---

## 1. 账号创建阶段

### TC-VW-01-01-01: 查看者账号创建 🔴

**目的**: 验证查看者账号创建流程正常

**前置条件**:
- Portal 服务已启动
- 已存在 viewer 角色

**测试步骤**:
```bash
# 由管理员创建
curl -X POST http://localhost:8010/api/users \
  -H "Authorization: Bearer $ADMIN_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "username": "viewer01",
    "password": "view123",
    "role": "viewer",
    "display_name": "查看者",
    "email": "viewer01@example.com"
  }'
```

**预期结果**:
- 状态码: 201
- 用户创建成功

---

### TC-VW-01-01-02: 查看者首次登录 🔴

**目的**: 验证查看者可以正常登录

**前置条件**:
- 查看者账号已创建

**测试步骤**:
```bash
curl -X POST http://localhost:8010/auth/login \
  -H "Content-Type: application/json" \
  -d '{"username": "viewer01", "password": "view123"}'
```

**预期结果**:
- 状态码: 200
- 响应包含:
  - `user.role`: "viewer"
  - `token`: 有效的 JWT token

---

### TC-VW-01-01-03: 查看者修改密码 🟡

**目的**: 验证查看者可以修改自己的密码

**前置条件**:
- 已登录查看者账号

**测试步骤**:
```bash
TOKEN=$(curl -s -X POST http://localhost:8010/auth/login \
  -H "Content-Type: application/json" \
  -d '{"username": "viewer01", "password": "view123"}' | jq -r '.token')

curl -X POST http://localhost:8010/auth/change-password \
  -H "Authorization: Bearer $TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "old_password": "view123",
    "new_password": "NewSecurePass123!"
  }'
```

**预期结果**:
- 状态码: 200
- 密码修改成功

---

### TC-VW-01-01-04: 验证查看者权限范围 🔴

**目的**: 验证查看者只拥有只读权限

**前置条件**:
- 已登录查看者账号

**测试步骤**:
```bash
TOKEN=$(curl -s -X POST http://localhost:8010/auth/login \
  -H "Content-Type: application/json" \
  -d '{"username": "viewer01", "password": "view123"}' | jq -r '.token')

curl -X GET http://localhost:8010/auth/permissions \
  -H "Authorization: Bearer $TOKEN"
```

**预期结果**:
- 状态码: 200
- 响应包含查看者权限:
  - `data:read`
  - `metadata:read`
  - 不包含 `data:write`, `data:delete`, `system:admin`

**目的**: 验证查看者权限被变更后生效

**前置条件**:
- 已登录查看者账号

**测试步骤**:
```bash

  -d '{"username": "viewer01", "password": "view123"}' | jq -r '.token')

# 权限撤销后尝试访问
curl -X GET http://localhost:8014/api/data/revoked_dataset \
  -H "Authorization: Bearer $TOKEN"
```

**预期结果**:
- 状态码: 403
- 响应包含: 权限不足

---

### TC-VW-07-01-02: 账号禁用 🟡

**目的**: 验证查看者账号被禁用后无法访问

**前置条件**:
- 查看者账号被管理员禁用

**测试步骤**:
```bash
TOKEN=$(curl -s -X POST http://localhost:8010/auth/login \
  -H "Content-Type: application/json" \
  -d '{"username": "viewer01", "password": "view123"}' | jq -r '.token')

# 账号禁用后使用旧 Token
curl -X GET http://localhost:8014/api/assets \
  -H "Authorization: Bearer $TOKEN"
```

**预期结果**:
- 状态码: 401 或 403
- 响应包含: 账号已被禁用或 Token 无效
